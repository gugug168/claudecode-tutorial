场景1：Excel 数据处理（20分钟）

任务：清洗销售数据

假设你有一个杂乱的销售数据表格 sales_data.xlsx，需要清洗。

bash 
claude ~/Documents

然后输入：

我有一个 sales_data.xlsx 文件，请帮我：
1. 读取这个文件
2. 去除重复行
3. 统一日期格式为 YYYY-MM-DD
4. 计算总销售额
5. 生成一个清洗后的文件和分析报告
🔍 Claude 会做什么？
  1. 读取 Excel 文件（用 Python 的 pandas 库）
  2. 写一个 Python 脚本清洗数据
  3. 计算统计数据
  4. 生成新的 Excel 文件 sales_data_cleaned.xlsx
  5. 生成一个分析报告 sales_report.md

提示词模板：Excel 数据清洗

📋 模板
提示词 
请帮我清洗 [文件名]：
1. 去除重复行
2. 统一日期格式为 YYYY-MM-DD
3. 空值处理：数值列用平均值填充，文本列用 "未知" 填充
4. 去除前后空格
5. 给出清洗前后对比

场景6：发票清单整理（15分钟）

任务：从杂乱的数据中提取发票信息

📋 场景说明

财务部门发来一堆发票数据，格式混乱，需要整理成标准的报销清单。

假设你有一个 invoices.xlsx 文件，数据格式如下：

日期 发票号码 金额(元) 类型
2024.01.15 12345678 ¥256.00 办公用品
2024/1/16 87654321 1280.50 差旅费
01-17-2024 11223344 ¥56.00 交通费

在 Claude Code 中输入：

请处理 invoices.xlsx 文件：

1. 统一日期格式为 YYYY-MM-DD
2. 统一金额格式（去掉 ¥ 符号，保留两位小数）
3. 添加"报销类别"列：办公用品→管理费用，差旅费→差旅费用，交通费→交通费用
4. 计算总金额
5. 生成一个新的报销清单.xlsx

提示词模板：发票整理

📋 模板
提示词 
请整理 [文件名] 发票数据：
1. 统一日期格式为 YYYY-MM-DD
2. 提取/清洗金额数据（数字格式）
3. 标准化发票号码格式
4. 按类型分类统计
5. 生成汇总报告

常见问题

Q1: Excel 文件太大怎么办？

可以告诉 Claude 分批次处理：

数据量较大，请：
1. 先分析数据结构（列名、类型）
2. 给出清洗方案
3. 分批次处理（每次 1000 行）
4. 最后合并结果
Q2: 文件整理会不会搞乱？

Claude 会先生成整理清单给你确认。你也可以让 Claude 只生成清单，不执行移动：

请只生成整理清单，不要移动文件
本章自检（3分钟）
点开问题查看答案，能讲清楚就算掌握。
用 Claude Code 处理 Excel/文件整理时，最先要说清楚的三件事是什么？
文件/范围是什么、你要的目标是什么、有哪些约束（不要改什么/输出什么格式）。
怎么避免“它一口气改太多，返工很痛”？
先让它给计划（或用 Plan 思路），确认后再执行；把任务拆小，一次只处理一个文件/一个目标。
批量移动/重命名之前，最安全的准备动作是什么？
先备份或 Git 存档；并在自己的练习/项目文件夹内操作，避免系统目录。
你希望最终交付物是什么，应该怎么写？
明确写“输出文件名/格式/内容”，例如生成 report.md、生成清洗后 CSV、生成归档清单。
