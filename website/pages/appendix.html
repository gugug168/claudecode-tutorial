A.7 常见问题速查

安装问题

问题 解决方案
npm 命令找不到 重新安装 Node.js，确保勾选 "Add to PATH"
Python 命令找不到 重新安装 Python，确保勾选 "Add Python to PATH"
claude 命令报错 检查 npm 全局路径，重新安装 Claude Code
Mac 提示权限不足 使用 sudo npm install -g @anthropic-ai/claude-code

使用问题

问题 解决方案
Claude 不回答 检查 API Key 配置，确认网络连接
不能修改文件 检查是否在只读模式，使用 /edit 退出
不知道当前模式 输入 /status 查看
如何退出 输入 /exit 或按 Ctrl+C

A.8 权限决策树

你要做什么？ │ ├─ 只是看看/学习 │ → claude --read-only │ 最安全，完全不会修改文件 │ ├─ 修改自己的文件 │ → claude （默认） │ 最常用，安全且有保护 │ ├─ 修改很多文件/批量操作 │ → claude --dangerous │ 方便但危险，要有备份 │ └─ 需要操作多个文件夹 → claude --allow-system-wide 权限很大，谨慎使用

A.11 进度管理

说明

进度只保存在你当前浏览器（localStorage）。换电脑/换浏览器会丢失，建议导出备份。

键盘快捷键

快捷键 功能
Ctrl + C退出 Claude Code
Ctrl + D退出 Claude Code
Ctrl + L清屏
↑ / ↓浏览历史命令

环境变量

变量 用途
ANTHROPIC_API_KEYAnthropic 官方 Key（部分版本/场景使用）
ANTHROPIC_AUTH_TOKENAnthropic 兼容 Token（国内转发/第三方常用，如智谱）
ANTHROPIC_BASE_URLAPI 端点地址（不同服务商不同）
API_TIMEOUT_MS超时（毫秒），网络慢时可适当调大
CLAUDE_CODE_DISABLE_NONESSENTIAL_TRAFFIC减少非必要网络请求（可选）

配置文件位置

配置 位置
全局配置~/.claude/settings.json
项目配置.claude/settings.json
MCP 配置~/.claude/mcp-servers.json
SKILLS 目录~/.claude/skills/
