P2-1：场景包（带练习素材）

随时练习 类别：附录
通过真实场景练习，快速掌握 Claude Code 的实际应用
这页怎么用？（新手最省心）
  1. 先选一个你最想练的场景。
  2. 下载练习素材（可用 Excel 直接打开 CSV）。
  3. 复制“一键提示词”，在 Claude Code 里执行。
  4. 对照“验收标准”检查结果。
S1 工程台账清洗（Excel）

去重、统一日期/单位、补齐缺失字段，输出干净的台账 + 清洗报告。

20分钟 简单
S2 资料归档（文件整理）

按“日期/类型/项目”自动建文件夹、移动文件、生成归档清单。

15分钟 简单
S3 日报/周报生成（模板化）

把零散记录整理成结构化日报/周报，输出 Markdown/Word 可复制版本。

15分钟 简单
提醒：先在“你自己的练习文件夹”里做

文件整理/批量改动前，建议先 Git 存档或复制一份文件夹，避免误操作。

遇到报错先看：A1：故障排查（按症状）
