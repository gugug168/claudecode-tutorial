A2：省钱与避坑

预计时间：10分钟 类别：附录
明确目标、分步执行，让AI帮你高效完成任务
新手最容易“烧钱/白忙”的 3 个原因
  1. 一上来就丢一大段内容，让模型反复猜。
  2. 目标不明确：让它“随便优化/随便看看”。
  3. 不给约束：它改得越多，返工越多。

三条省钱铁律（照做就省）

  1. 先小后大：先用 1 个最小例子跑通，再把完整文件交给它。
  2. 先问后做：先让 Claude 问你 3 个关键问题，确认需求再执行。
  3. 先定边界：明确“允许改什么 / 不允许改什么 / 输出格式”。

大文件/长对话怎么处理？（避免超长失败）

  • 把任务拆成 3 步：提取 → 清洗 → 输出。
  • 一次只处理一个文件/一个函数/一个表。
  • 让 Claude 先给“计划 + 你确认后再执行”。

万能省钱提示词模板（直接复制）

模板1：先问清楚再动手（最省）

提示词 
我有一个任务：[一句话说明]
在你开始做之前，请先问我 3 个你最需要确认的问题（只问问题，先别执行）。

模板2：只改最少的地方（减少返工）

提示词 
请修改 [文件名/内容]，目标是：[目标]
约束：
1) 不要改动无关内容
2) 如果需要新增文件/删除文件，先征求我确认
3) 输出：只告诉我“改了什么 + 为什么”

模板3：分三步处理数据（稳定又省）

提示词 
请按 3 步完成：
1) 先告诉我你准备怎么做（计划）
2) 我确认后再执行
3) 执行后给我结果 + 可复用的命令/脚本
最关键的一句话

你越"说清楚"，就越省钱；你越"让它猜"，就越烧钱。
