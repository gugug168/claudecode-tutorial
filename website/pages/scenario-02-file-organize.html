S2：资料归档（文件整理）

预计时间：15分钟 难度：简单 类型：练习场景
Claude Code 可以理解并整理文件结构
强烈建议：先在"练习文件夹"里做

这个场景会移动/重命名文件。不要在真实资料根目录直接做。

练习素材

下载并解压到你自己的练习文件夹中：

下载：s2_资料归档_练习.zip

提示

这是一些“假文件”（内容不重要，文件名才重要），用于练习日期识别与归档分类。

解压后建议先不要改文件名，跑通一次再换成你自己的真实资料。

练习步骤

  1. 新建文件夹：资料归档练习
  2. 把上面的压缩包解压到这个文件夹里（会得到一堆带日期/类型的文件）。
  3. 在该文件夹运行 claude，复制下面提示词。

一键提示词（直接复制）

提示词 
请整理当前文件夹里的所有文件，目标是形成清晰的归档结构。

规则：
1) 按类型建文件夹：01_会议纪要、02_图纸、03_照片、04_报表、05_合同、99_其他
2) 如果文件名里包含日期（YYYY-MM-DD 或 YYYYMMDD），则在对应类型目录下再按月份建子目录（例如：2026-01）
3) 移动文件到对应目录（不要删除文件）
4) 生成归档清单 archive_index.md，内容包括：原文件名、目标路径、识别出的日期/类型依据

执行前：
先输出你准备怎么分类的“规则表”，我确认后再开始移动。

验收标准

完成即合格
