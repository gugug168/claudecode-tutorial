为什么学习 Claude Code？

01 自动化日常任务

自动处理 Excel 数据、整理文件、生成报告， 让你从重复劳动中解放出来。

02 零编程基础也能用

不需要学习复杂的编程语法， 用自然语言就能告诉 AI 你想做什么。

03 适合土木专业人士

专门面向非 IT 人士设计， 案例和练习都贴近日常工作场景。

04 中文友好

支持国产 AI 模型（智谱、豆包等）， 中文交互体验更好。

开始学习前的准备

你需要：

  • 1 一台 Windows 或 Mac 电脑
  • 2 约 6 小时的学习时间（可分多次完成）
  • 3 能使用电脑的基本操作（打字、打开文件）

常见问题

Q: 我完全不会编程，能学吗？
完全可以！本教程就是面向零基础人群设计的。你只需要会用电脑打字、浏览网页就行。
Q: Claude Code 是免费的吗？
Claude Code CLI 本身是免费的，但需要连接 AI 模型才能使用（模型调用通常会产生费用/额度消耗）。本教程目前开放阅读，后续可能调整为收费或部分内容收费。
Q: 学完需要多久？
必修课程约 3-4 小时，全部课程约 6 小时。你可以按自己的节奏学习，随时可以暂停和继续。
Q: 需要安装什么软件？
需要安装 Node.js 和 Claude Code CLI（必需）。Python 属于“可选项”（做 Excel/数据脚本时再装也来得及）。第一章会详细教你如何安装，一步步跟着做就行。
